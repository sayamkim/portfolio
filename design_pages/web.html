portfolio

I chose to build a portfolio site from scratch to explore front end tools and processes. Using a template would have saved time and simplified the process but I wanted to make sure that the site would be an opportunity to take a deeper look into the building blocks of the web.

Producing custom design assets was also important as I wanted to approach the project with a holistic workflow that would produce insight into both development and design.

The site has gone through a number of iterations incorporating lessons learned along the way and I plan on using it as an ongoing platform for code and design experiments. Stay tuned!

space
